# Python Character Manager

## Overview
This project is a command-line Python application that manages a list of characters (heroes and villains).

It was developed as part of a university programming assignment and demonstrates core programming concepts such as file handling, data processing, and algorithm design.

---

## Project Context
This project was completed as part of the **Problem Solving and Programming** course at university.

The assignment required building a program to manage character data, including reading from and writing to files, handling user input, and implementing logic for sorting, searching, and simulation.

---

## Features

- Load character data from file  
- Add and remove characters  
- Search characters  
- Simulate battles between characters  
- Track wins, losses, and health  
- Sort characters by health  
- Save updated data to file  

---

## Technologies Used

- Python  
- File handling (read/write)  
- Lists and data structures  
- Functions and modular programming  

---

## How to Run

1. Make sure `characters.txt` is in the same folder  
2. Run:
python main.py
